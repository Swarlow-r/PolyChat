## 如何使用（安装说明）

你需要的是“加载未打包的扩展程序”（也就是从源码/解压目录直接加载）。

1. **下载并解压** PolyChat 的压缩包到你本地的某个文件夹（确保里面能看到 `manifest.json` 等扩展文件）。
2. 打开浏览器的扩展管理页：
  - Chrome：`chrome://extensions`
  - Edge：`edge://extensions`
3. 打开**开发者模式**。
4. 点击 **“加载已解压的扩展程序 / Load unpacked”**。
5. 选择你刚才解压出来的那个目录（不要选到某个子文件夹，直接选包含扩展文件的根目录）。
6. 安装完成后，PolyChat 的扩展会出现在浏览器里，按你使用的 AI 网站进入对话页即可开始使用。

如果后续功能没有生效，通常是扩展未加载成功或目标网站改版导致匹配规则变化，你可以尝试重新“加载已解压的扩展程序”。

## 隐私与数据

PolyChat 会把你在网页中的对话内容相关信息**保存到你的浏览器本地**用于搜索与回溯。它不需要你把账号密码交给插件；具体数据是否会被 AI 网站本身继续处理，仍以各 AI 平台的隐私与规则为准。
